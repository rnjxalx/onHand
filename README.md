# OnHand — 반려묘 케어 앱 (React Native / Expo)

가족이 함께 고양이를 돌보는 반려묘 케어 앱입니다.
할 일·일정·건강 기록을 가족끼리 실시간으로 공유하고, 고양이 캐릭터가
오늘의 케어 상태에 따라 표정으로 반응합니다.

## 주요 기능

- **홈** — 고양이 캐릭터가 할 일·기분에 따라 기본/행복/슬픔 표정으로 바뀜, 외출·귀가 인사
- **케어** — 가족과 공유하는 할 일 체크리스트, 가족 채팅, 관리(발톱·샤워·빗질) 주기 관리
- **캘린더** — 일정 등록·확인, 활동 피드
- **MY** — 고양이 프로필, 예방접종·체중 기록, 추억 사진
- **가족 공유** — 초대 코드로 가족에 합류, 할 일·고양이 정보·채팅 실시간 동기화
- **알림** — 목표 시간을 정한 케어 항목에 30분 전 알림
- **AI 캐릭터** — 고양이 사진으로 basic / happy / sad 캐릭터 자동 생성

## 실행 방법 (데모)

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 시작
npx expo start
```

터미널에 QR코드가 나타납니다.

- **실제 휴대폰**: App Store / Play Store에서 **Expo Go** 앱을 설치한 뒤 QR코드 스캔
- **iOS 시뮬레이터**: 터미널에서 `i` 입력 (Mac + Xcode 필요)
- **Android 에뮬레이터**: 터미널에서 `a` 입력 (Android Studio 필요)
- **웹 브라우저**: 터미널에서 `w` 입력
